# my-works-hub
个人作品集合，提供下载/图片/使用说明等


图片，截图，使用视频，这个相对固定，一行展示，左右滑动。
开源地址
下载链接，返回一个zip。
使用教程


一个根文件夹resources，一个项目一个文件夹，
media：图片/视频，按照1/2/3/4有序排列，前端也按顺序展示，存放效果图/使用说明/视频展示/视频教程等
zip ：存放zip文件

数据库
项目中文名，git开源地址(可空)，文字教程，标签（网站/浏览器脚本），项目文件夹名，项目开发时间，观看次数，下载次数
